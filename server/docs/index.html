Utils
  • define
    Example: rodrigo define «word»
    Result: Embed with the urban dictionary definition of the word
  • search
    Example: rodrigo search «word»
    Result: Embed with the first 5 google results
  • math
    Example: rodrigo math 1+1
    Result: 2
  • sort
    Example: rodrigo sort randomly;words;sorted
    Result: words > randomly > sorted
  • convert
    Example: rodrigo convert 50 USD to EUR
    Result: 44.81
  • weather
    Example: rodrigo weather «location»
    Result: Embed with the weather for that location
  • remindme
    Example: rodrigo remindme of «something» in «number» «minutes/hours»
    Result: After the requested time the bot will return the message

Youtube
  • youtube
    Example: rodrigo youtube «youtube channel name»
    Result: Embed with the last youtube video from that channel
  • add
    Example: rodrigo youtube add «youtube channel name»
    Result: Channel is added to notifications
  • remove
    Example: rodrigo youtube remove «youtube channel name»
    Result: Channel is removed from notifications
  • get
    Example: rodrigo youtube get
    Result: List of all the youtube channels that will notify the configured text channel when there are new videos
